Leaper Vision Toolkit

高性能 Leaper 机器视觉工具库

Leaper Vision Toolkit (LPV) 为帮助用户构建应用于各个工业检测领域的机器视觉应用而设计，并提供最大程度的灵活性。LPV 允许开发者按照实际需求任意组织视觉处理流程，配置多线程，调整任意视觉算法工具的所有开放参数。

完全掌控整个机器视觉系统

LPV 提供稳定、有效的 Leaper 视觉算法工具库，用户可自由构建应用于各个领域的机器视觉解决方案，包括定位、测量、对位、检测和识别等。

高效 · 灵活 · 可扩展性

为快速设计和实施而设计，满足 OEM 和高级机器视觉系统开发的需求。多种适用不同使用场景的算法库和用户交互界面控件，可完全整合 LPV 模块到 OEM 应用及设备平台。

连接到灵闪 IntelliBlink 平台

灵闪是一个基于利珀科技机器视觉基础框架开发的轻便的智能算法配置及组合工具平台。通过所见即所得的交互界面和简单的拖拽操作，用于快速开发和迭代定位、测量、缺陷检测等机器视觉任务。灵闪平台提供一系列覆盖具体应用需求的算法工具包。LPV 能够无缝连接到灵闪平台，帮助用户直接使用其在灵闪平台上开发的机器视觉解决方案，将其集成到 LPV 应用程序中运行、操作，实现更大程度上的灵活性和可扩展性。

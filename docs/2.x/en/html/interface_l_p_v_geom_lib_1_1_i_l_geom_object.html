)

Detect if the cursor is placed over one of the dragging handles.
For most primitives, only LPVGeomHandleInside is supported, including:
point, circle, rectangle, ellipse, and arc.
For a line and polygon object, LPVGeomHandleInside is for updating placement of the line while LPVGeomHandleMoveVetex is for moving its vertex.
For a points object, when hit on any point in the collection, LPVGeomHandleMoveVertex handle is returned if enabled for moving the single point, otherwise LPVGeomHandleInsid is returned for moving the whole collection.

Parameters
[in]posXThe x-coordinate of the cursor position.
[in]posYThe y-coordinate of the cursor position.
Return values
geomHandleReturn a handle identifier.

Draw(), Drag().

Remarks
This function is currently not available on Linux / MacOS platform.

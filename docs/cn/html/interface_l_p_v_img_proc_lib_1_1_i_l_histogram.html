详细描述

该接口类提供了计算图像直方图的功能。

图像直方图将输入图像中的像素按照其灰度值，统计其数值在给定范围和步长上的分布。 在一个视觉系统中，图像直方图通常用于观察和分析输入图像的亮度和对比度状态，用于调整整个视觉系统的光学成像。 也可根据直方图中隐含的灰度特征，用于目标识别和检测。

使用这个接口，需要创建一个 LHistogram 对象。

C++:

CLHistogram m_hist;
m_hist.Build(m_img, m_roi, 256, 0, 255);

C#:

LHistogram m_hist = new LHistogram();
m_hist.Build(m_img, m_roi, 256, 0, 255);

[in, defaultvalue(256)] int  binCount,
[in, defaultvalue(0)] int  lowerBound,
[in, defaultvalue(255)] int  upperBound 
)

基于输入图像和 ROI 区域，生成直方图。

参数
pImg输入图像
pRegion输入 ROI 区域，可为空（检测全图）
binCount将直方图分为若干个统计区域，推荐设置为总数据范围的因子，比如对一个灰度图，常用设置为 256 ，128，32 等。
lowerBound统计数据范围的下限，比如对一个灰度图，常用设置为 0.
upperBound统计数据范围的上限，比如对一个灰度图，常用设置为 255.

)

将直方图绘制到给定的设备。

参数
hdc绘制设备，可为窗口、位图等的上下文句柄。
bDrawFlags用于控制绘制的内容，查看 LPVChartDrawFlags。
xGridStep绘制 x 轴（索引）的步长，默认设置为 0，表示按照直方图数据自动选择合适的步长。
yGridStep绘制 y 轴（数值）的步长，默认设置为 0，表示按照直方图数据自动选择合适的步长。

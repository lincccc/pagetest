详细描述

该接口对应整个灵闪作业的运行结果，其中包含该作业中所有算法工具的输出结果。

通常通过 ILIBService::ExecuteTaskSync 得到该 LIBTaskResult 对象。 可以通过该接口访问单个算法工具的具体输出结果。

C++:

CLIBTaskResult taskResult;
long err = m_ib_service.ExecuteTaskSync(m_ib_task_id, image, (LPDISPATCH*)&taskResult);
if (err != LPVErrorCode::LPVNoError) return; // execution failed

C#:

LIBTaskResult taskResult;
int err = m_ib_service.ExecuteTaskSync(m_ib_task_id, image, out taskResult);
if (err != (int)LPVErrorCode.LPVNoError) return; // execution failed

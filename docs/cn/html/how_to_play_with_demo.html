如何操作 DEMO 程序

DEMO 程序位于 demo 文件夹下的所需平台对应的文件夹内，如 demo/x64 。

注意：DEMO 程序的平台需与安装注册的 lpv 版本对应。

DEMO 程序

Sanity DEMO

演示图像、ROI、模板、标定等基础功能。

源代码位置（C++）：mfcdemo/MFCSanity

源代码位置（C::）：csharpdemo/CSharpSanity

  1. 点击 Load Image 按钮，加载图片和模板。
  2. 点击 Template Learn 按钮，按照roi的位置抠图，学习模板。
  3. 点击 Template Match 按钮，执行模板匹配，右侧显示像素坐标。
  4. 点击 Calibrate 按钮，执行9点标定，右侧显示物理坐标。
  5. 点击 Reset Calibration 按钮，重置标定，右侧显示像素坐标。
  6. 点击 Save Calibration 按钮，保存标定文件，右侧显示物理坐标。
  7. 点击 Load Calibration 按钮，加载标定文件。
  8. 点击 Copy & Save 按钮，执行保存图像，在 exe 同级的 images 目录保存拷贝的 jpg 图像。
  9. 拖动图形界面中的 ROI 边界，可移动或改变 ROI 形状。
  10. 点击 Reset Roi 按钮，按照当前图像尺寸重置roi位置。

HikCam DEMO

演示相机与图像类间的交互，以海康相机为例。

源代码位置（C::）：csharpdemo/CSharpHikCam

  1. 安装海康相机驱动。
  2. 打开 MVS 查询相机序列号，填入右侧文本输入框中。
  3. 点击 Start 按钮，打开相机。
  4. 点击 Run (Continues Trigger) 按钮，切换为连续触发，不断更新和显示相机采集到的图像。
  5. 点击 Snap (Soft Trigger) 按钮，切换为软触发，获取一帧图像并显示。

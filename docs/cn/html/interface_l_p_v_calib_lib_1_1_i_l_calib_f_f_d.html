详细描述

该接口对应一个 FFD 相机标定模型。

在 FFD 标定模型中，使用自由形变网格来描述图像和物理坐标之间的映射关系。形变网格为参数化的自由曲面，基于提供的图像和物理点对优化得到。
该模型适应于任意的畸变类型。

使用这个接口，需要创建一个 LCalibFFD 对象。

C++:

CLCalibFFD m_calib;
m_calib_board.Detect(m_img, (LPDISPATCH*)&m_pnt_pairs);
m_calib.AddPoints(m_pnt_pairs);
m_calib.SetFFDGrid(4, 4); // use 4x4 FFD grid
LPVErrorCode err = (LPVErrorCode)m_calib.Calibrate();
if (err != LPVErrorCode::LPVNoError || !m_calib.IsCalibrated()) return;

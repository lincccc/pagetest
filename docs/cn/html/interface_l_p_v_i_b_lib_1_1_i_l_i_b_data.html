BSTR Name [get]
 数据对象的本地化名称。
对于输出的结果数据，则为输出结果的名称。对于输入的参数数据，则为参数的名称。
本地区域需在灵闪平台的配置中配置。
 

详细描述

该接口对应一个来自灵闪服务的数据，可能是输入参数数据或输出的结果数据。

C++:

// as result
CLIBData ibDataResult = algoResult.get_Result(L"Line"));
CLLine line = ibDataResult.ToLine();
// as input
CLIBData ibDataInput;
ibDataInput.FromDouble(12.34);
algoInputs.put_Config(L"Score", ibDataInput);

C#:

// as result
LIBData ibDataResult = algoResult.Result["Line"];
LLine line = ibDataResult.ToLine();
// as input
LIBData ibDataInput = new LIBData();
ibDataInput.FromDouble(12.34);
algoInputs.Config["Score"] = ibDataInput;

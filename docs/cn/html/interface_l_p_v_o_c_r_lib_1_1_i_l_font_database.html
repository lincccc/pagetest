属性

BSTR Name [get]
 样本库的标识符，通常为全局唯一的，用于区分和标识该数据库。 注意：样本库的标识符可能与分类的标签名一致，也可以不一致。
 
BSTR WorkingDir [get]
 样本库的工作目录。 对于持久化数据库，该工作目录用于存储样本文件和管理信息文件。对于内存数据库，该工作目录为固定的 "in-memory" 文本。
 
BOOL UseCache [get, set]
 样本的图像和区域对象是否保持缓存

◆ Create()

void Create ( BSTR  fontName,
BSTR  workingDirectory 
)

创建和初始化样本库。

参数
[in]labelName样本库的标识符
[in]workingDirectory工作目录路径。 对于内存数据库，设置工作目录为空。 对于其他非空路径，将创建一个持久化数据库，并在该目录下初始化一个空数据库。因此，该工作目录需为一个可读可写的路径。 若该目录下已存在一个样本数据库，将优先加载该数据库，标识符将可能被替换为加载的名称。

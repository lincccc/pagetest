详细描述

该接口提供斑点分析的功能。

斑点分析是机器视觉系统中常见的前景目标分割方法，用于从背景中提取灰度值近似的连通域生成斑点对象。 再计算斑点对象的属性特征，典型的几何特征如面积、轮廓周长、圆度等，图像特征如平均亮度、对比度等， 可用于定位、测量、缺陷检测等大多数典型视觉应用场景。

C++:

CLBlobAnalysis m_blob_ana;
m_blob_ana.AddBlobRange(50, 100);
LPVErrorCode err = (LPVErrorCode)m_blob_ana.Build(m_img, m_roi, (LPDISPATCH*)&m_blob_result);
if (err != LPVErrorCode.LPVNoError) return;

基于输入的图像和给定的像素坐标，自动生成一个灰度值范围，可用于添加为前景或背景范围。

参数
pImage输入的图像

y鼠标位置（选择中的）像素的 y 坐标

返回值

参数

zoomYy 方向的缩放系数，默认设置为 0，即与 x 方向的缩放系数相同，也就是等比缩放。
panXx 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)
panYy 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)

◆ SetRangeThresholdAdaptLocal()

HRESULT SetRangeThresholdAdaptLocal ( [in] enum LPVPolarity  blobPolarity,
[in, range(0, 999)] int  blockSize,
[in, defaultvalue(0), range(-255, 255)] int  bias 
)

使用局部自适应阈值用于分割。自动基于每个像素所在的 \(blockWidth \times blockHeight\) 范围内的像素灰度值分布，为每个位置计算一个合适的阈值，再减去 bias 作为最终的阈值。

参数
blobPolarityLPVBlackOnWhite 表示灰度值小于全局自适应阈值的像素组成斑点对象, LPVWhiteOnBlack 则相反。
blockSize局部区域的尺寸，需要是一个奇数，如 3, 5, 7 等。
bias基于该位置的局部自适应阈值计算，用于生成阈值 \( t' = t - bias \).

◆ SetSoftThresholdRange()

HRESULT SetSoftThresholdRange ( [in, range(0, 255)] int  softRange)

设置软阈值范围。默认值为 0，表示不开启软阈值功能。
用于提高斑点定位的鲁棒性和角度。
比如，若斑点的灰度值范围为 [100, 200]，软阈值范围为 5，则 [0, 95] 和 [205, 255] 为背景范围，[105, 195] 为前景斑点范围， [96, 104] 的范围按照 0.1 ~ 0.9 的权重属于前景斑点的范围，[196, 204] 的范围按照 0.9 ~ 0.1 的权重属于前景斑点的范围。 软阈值配置后，斑点分析得到斑点对象将使用软蒙版表示前景，软蒙版中的前景像素基于其权重值影响斑点的几何和图像属性计算，如重心等。

参数
softRange软阈值范围，生成 [t - softRange, t + softRange] 范围。

◆ Draw()

HRESULT Draw ( [in] VARIANT_BOOL  bFill,
[in] HDC  hdc,
[in, defaultvalue(1)] double  zoomX,
[in, defaultvalue(0)] double  zoomY,
[in, defaultvalue(0)] double  panX,
[in, defaultvalue(0)] double  panY 
)

绘制多边形到给定的设备上

参数
bFill是否使用当前笔刷填充该多边形
hdc绘制设备，可为窗口、位图等的上下文句柄。
zoomXx 方向的缩放系数，默认为真实尺寸 100。
zoomYy 方向的缩放系数，默认设置为 0，即与 x 方向的缩放系数相同，也就是等比缩放。
panXx 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)
panYy 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)

◆ IntersectLine()

HRESULT IntersectLine ( [in] ILLine *  pLine,
[out] ILPoints **  ppIntersectPoints,
[out, retval] LPVIntersectType *  pType 
)

判断当前多边形和输入直线是否相交，若相交，返回一个或多个交点。

参数
pLine输入的直线
返回值
ppIntersectPoints返回交点的点集，若当前多边形和输入直线不相交，则该点集为空，否则可能包含一个或多个点。
pType返回相交的类型，与交点中在直线线段上的个数相关， Bounded (所有), Partial (部分) or Unbounded (无).
参见
ILLine::IntersectPoly().

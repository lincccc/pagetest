详细描述

该接口对应一个圆形区域对象。

使用这个接口，需要创建一个 LCircleRegion 对象。

示例代码

C++
ILCircleRegionPtr m_circle_roi(__uuidof(LCircleRegion));
m_circle_roi->SetPlacement(20, 20, 100);
C#
LCircleRegion m_circle_roi = new LCircleRegion();
m_circle_roi.SetPlacement(20, 20, 100);

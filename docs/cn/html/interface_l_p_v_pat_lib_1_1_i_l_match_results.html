详细描述

该接口提供一组模板匹配结果的集合，包含一个或多个模板匹配结果。

通常通过调用 ILMatch::Match() 得到该检测结果集合。 可以通过该接口访问其中的单个检测结果，绘制检测结果到给定的设备。

C++:

CLMatchResults m_match_results;
LPVErrorCode err = (LPVErrorCode)m_match.Match(m_img, m_roi, (LPDISPATCH*)&m_match_results);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_match_results->Count();
for (int i = 0; i < c; ++i) {
LMatchResult result = m_match_results->Item(i);
// handle each line result
}

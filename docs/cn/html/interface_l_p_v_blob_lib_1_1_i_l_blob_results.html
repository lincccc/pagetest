详细描述

该接口提供一组斑点分析结果的集合，包含一个或多个斑点。

通常通过调用 ILBlobAnalysis::Build() 得到该检测结果集合。 可以通过该接口访问其中的单个检测结果，绘制检测结果到给定的设备。

C++:

CLBlobResults m_blob_results;
LPVErrorCode err = (LPVErrorCode)m_blob_ana.Build(m_img, m_roi, (LPDISPATCH*)&m_blob_result);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_blob_results->Count();
for (int i = 0; i < c; ++i) {
CLBlob blob;
blob = m_blob_results->Item(i);
// handle each blob result
}

详细描述

该接口类提供了多种形态学算子。

形态学算子通常使用一个预设的结构元，如矩形、圆形或十字等。对输入图像中的每个像素，形态学算子将这个像素的灰度值与其周围像素的灰度值进行比较， 将比较结果输出到输出图像的相应位置。结构元定义了像素周围需要比较的的领域范围。 膨胀和腐蚀是最基础的形态学算子，其他算子由这两个算子组合得到。膨胀将比较的最大值作为结果，而腐蚀取比较的最小值。

使用这个接口，需要创建一个 LImageMorph 对象。

C++:

CLImageMorph m_morph_op;
m_morph_op.SetMorphShape(LPVMorphRect, 3, 3);
m_morph_op.Close(m_img, m_result_img);

详细描述

LPV 高级定位模块，提供用于复杂目标或边缘的定位算法。

该模块提供一些算法类，用于目标或边缘定位的复杂任务。使用该模块中的类和算法，需按照以下方式引用该模块：

示例代码

C++
#include "LPVLocateX.h"
C#
using LPVLocateXLib;

枚举值

LPVPathDrawEndPoints

LPVPathDrawIndex

LPVPathDrawDefault

LPVPathDrawAll

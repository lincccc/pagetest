详细描述

该接口提供连接到灵闪服务的功能。

灵闪是一个基于利珀科技机器视觉基础框架开发的轻便的智能算法配置及组合工具平台。 通过所见即所得的交互界面和简单的拖拽操作，用于快速开发和迭代定位、测量、缺陷检测等机器视觉任务。 灵闪平台提供一系列覆盖具体应用需求的算法工具包: 特征定位、数量检测、有无判定、基础计算、图像变换、结合测量和计算、相机标定、符号识别、物体分类等等。

LPV IB 服务模块帮助用户在基于灵闪快速编辑机器视觉方案的基础上，将其集成到 LPV 应用程序中运行、操作， 实现更大程度上的灵活性和可扩展性。

C++:

CLIBService m_ib_service;
m_ib_service.SetUp(L"path to the IntelliBlink root folder, say D:/Programs/IntelliBlink/");
if (m_ib_service.IsReady()) return; // set up IB service failed
// do stuff, like load task, execute then get result
m_ib_service.TearDown();

C#:

LIBService m_ib_service = new LIBService();
m_ib_service.SetUp("path to the IntelliBlink root folder, say D:/Programs/IntelliBlink/");
if (m_ib_service.IsReady()) return; // set up IB service failed
// do stuff, like load task, execute then get result
m_ib_service.TearDown();

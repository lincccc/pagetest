详细描述

LPV 定位模块，提供经典的目标或边缘的定位算法。

该模块提供一些算法类，用于经典的目标或边缘定位任务。使用该模块中的类和算法，需按照以下方式引用该模块：

C++:

#include "LPVLocateLib.h"

C#:

using LPVLocateLib;

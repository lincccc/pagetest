详细描述

该接口提供条码检测和质量评价的功能。

支持大多数流行的一维和二维码，具体列表请查看 LPVBarcodeType。 质量评价符合 ISO-15416（一维码）和 ISO-15415（二维码）标准。

C++:

CLBarcodeDetector m_barcode_detector;
m_barcode_detector.put_Type(LPVBarcodeType::LPVBarcodeAll1D); // detect all types of 1D barcode
m_barcode_detector.put_VerifyQuality(VARIANT_TRUE); // do quality verification during detection
m_barcode_detector.put_MaxCount(3);
LPVErrorCode err = (LPVErrorCode)m_barcode_detector.Detect(m_img, m_roi, (LPDISPATCH*)&m_code_results);
if (err != LPVErrorCode.LPVNoError) return;

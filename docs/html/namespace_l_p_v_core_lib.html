详细描述

LPV 基础模块，提供图像、ROI、绘制等基础功能。

所有其他 LPV 模块均依赖该基础模块。使用该模块中的类和算法，需按照以下方式引用该模块：

C++:

#include "LPVCore.h"

C#:

using LPVCoreLib;
LPVCoreLib
LPV 基础模块，提供图像、ROI、绘制等基础功能。

MFC Demo: MFCSanity, MFCRegion
C# Demo: CSharpSanity, CSharpRegion, CSharpHikCam

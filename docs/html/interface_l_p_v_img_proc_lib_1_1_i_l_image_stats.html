详细描述

该接口类提供了用于图像的各种属性统计的算子。

使用这个接口，需要创建一个 LImageStats 对象。

C++:

CLImageStats m_img_stats;
m_img_stats.MinMax();

C#:

LImageStats m_img_stats = new LImageStats();
m_img_stats.MinMax();

计算输入图像的灰度均值和标准差

参数
pImg输入图像。
pRegion输入 ROI 区域，可为空。
lb阈值范围下限，数值在 0 ~ 255 范围内
ub阈值范围上限，数值在 0 ~ 255 范围内
返回值
pBelowCount输出灰度值在阈值范围以下的像素总数。
pBetweenCount输出灰度值在阈值范围之间的像素总数。
pAboveCount输出灰度值在阈值范围以上的像素总数。

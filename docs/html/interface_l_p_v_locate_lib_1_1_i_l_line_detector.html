详细描述

该接口提供直线定位的功能。

直线定位用于在输入图像中和指定的区域中，定位一条或多条直线边缘。
该算法具有高实时性、高亚像素精度等特点，其局限性在于其定位直线的角度在给定区域的扫描方向的 (-10, +10) 范围内。 因此，使用该算法需要指定目标直线的大致方向。

C++:

CLLineDetector m_line_detector;
m_line_detector.put_Polarity(LPVPolarity::LPVPolarityEither);
LPVErrorCode err = (LPVErrorCode)m_line_detector.Detect(m_img, m_roi, (LPDISPATCH*)&m_line_results);
if (err != LPVErrorCode.LPVNoError) return;

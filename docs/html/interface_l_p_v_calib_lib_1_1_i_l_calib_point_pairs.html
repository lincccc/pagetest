详细描述

该接口对应一组图像和物理点对的集合。

通常通过调用 ILCalibBoard::Detect() ，由标定板检测和生成该点对集合，也可以通过 AddPointPair() 手动构建点对集合。 可以通过该接口访问其中的单个检测结果，绘制检测结果到给定的设备。

C++:

CLCalibPointPairs m_pnt_pairs;
LPVErrorCode err = (LPVErrorCode)m_calib_board.Detect(m_img, (LPDISPATCH*)&m_pnt_pairs);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_pnt_pairs->Count();
for (int i = 0; i < c; ++i) {
LPoint imgPnt = m_pnt_pairs->GetImagePoint(i);
LPoint worldPnt = m_pnt_pairs->GetWorldPoint(i);
// handle each point pair
}

)

绘制索引对应的单个点对中的图像点到给定的设备。

参数
index输入索引，从 0 开始但小于总数的数字
hdc绘制设备，可为窗口、位图等的上下文句柄。
zoomXx 方向的缩放系数，默认为真实尺寸 100。
zoomYy 方向的缩放系数，默认设置为 0，即与 x 方向的缩放系数相同，也就是等比缩放。
panXx 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)
panYy 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)

详细描述

该接口对应一个自定义标定模型。

自定义标定模式是一个简易的完全由用户提供参数的标定模型，使用于简单的低精度的已知物理和图像坐标系间的缩放系数、平移量和旋转角度的场景。
该标定模型不支持镜头畸变。

使用这个接口，需要创建一个 LCalibCustom 对象。

C++:

CLCalibCustom m_calib;
// create the coordinates at the center of the image
m_calib.put_AxisDirX(LPVAxisRight);
m_calib.put_AxisDirY(LPVAxisUp);
m_calib.MoveOriginTo(w / 2, h / 2);
LPVErrorCode err = (LPVErrorCode)m_calib.Calibrate();
if (err != LPVErrorCode::LPVNoError || !m_calib.IsCalibrated()) return;

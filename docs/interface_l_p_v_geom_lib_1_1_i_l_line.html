)

绘制直线到给定的设备

参数
hdc绘制设备，可为窗口、位图等的上下文句柄。
zoomXx 方向的缩放系数，默认为真实尺寸 100。
zoomYy 方向的缩放系数，默认设置为 0，即与 x 方向的缩放系数相同，也就是等比缩放。
panXx 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)
panYy 方向的平移。最终的 x 坐标按照该公式计算: \((x + panX) \times zoomX\)

◆ IntersectCircle()

HRESULT IntersectCircle ( [in] ILCircle *  pCircle,
[out] ILPoints **  ppIntersectPoints,
[out, retval] LPVIntersectType *  pType 
)

判断当前直线和输入圆形是否相交，若相交，返回 1 或 2 个交点。

参数
pCircle输入的圆形
返回值
ppIntersectPoints返回交点的点集，若当前圆形和输入直线不相交，则该点集为空，若当前圆形和输入直线相切，返回 1 个交点， 若当前圆形和输入直线相交，返回 2 个交点。
pType返回相交的类型，与交点的个数相关， Bounded (2), Partial (1) or Unbounded (0).
参见
ILCircle::IntersectLine().

Detailed Description

This interface expose APIs to create/modify an annulus region object.

To use this interface, you should create a LAnnulusRegion object.

Example Code

C++
ILAnnulusRegionPtr m_annulus_roi(__uuidof(LAnnulusRegion));
m_annulus_roi->SetPlacement(20, 20, 100, 200);
C#
LAnnulusRegion m_annulus_roi = new LAnnulusRegion();
m_annulus_roi.SetPlacement(20, 20, 100, 200);

Detailed Description

This interface expose APIs to create/modify/manipulate/draw a LPV region object (LRegion , LRectRegion , LRotRectRegion , LCircleRegion , etc.).

Region object is useful to restrict the detection and inspection to required ROI. To use this interface, you should create a region object. There are several ways to modify a region's placement. The most popular options are listed below:

  1. For simple geometric region, do it programmatically.
    m_rect_roi.SetPlacement(10, 10, 200, 100);
  2. Interactive with the region via cursor, by dragging the handles.
    m_move_handle = (LPVRoiHandle)m_roi.HitTest(cursor_x, cursor_y, zoomx, zoomy);
    m_roi.Drag(m_move_handle, cursor_x, cursor_y, zoomx, zoomy);
    LPVCoreLib::LPVRoiHandle
    LPVRoiHandle
    This enumeration represents the type of a region handle for drag.
    Definition: LPVCore.idl:308
  3. Create a new region via logical operators on multiple existing regions.
    CLRegion new_roi = m_rect_roi.Add(m_rect_roi_other);

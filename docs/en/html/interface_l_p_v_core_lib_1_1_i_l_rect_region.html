Detailed Description

This interface expose APIs to create/modify a rectangle region object.

To use this interface, you should create a LRectRegion object.

Example Code

C++
ILRectRegionPtr m_rect_roi(__uuidof(LRectRegion));
m_rect_roi->SetPlacement(20, 20, 100, 50);
C#
LRectRegion m_rect_roi = new LRectRegion();
m_rect_roi.SetPlacement(20, 20, 100, 50);

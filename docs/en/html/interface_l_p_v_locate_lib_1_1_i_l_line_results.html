Detailed Description

This interface holds a collection of line detection results.

Usually you get the LLineResults object from the ILLineDetector::Detect() as detection result. You can then use this interface to access individual result and draw the results on graphics windows.

C++:

CLLineResults m_line_results;
LPVErrorCode err = (LPVErrorCode)m_line_detector.Detect(m_img, m_roi, (LPDISPATCH*)&m_line_results);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_line_results->Count();
for (int i = 0; i < c; ++i) {
LLineResult result = m_line_results->Item(i);
// handle each line result
}

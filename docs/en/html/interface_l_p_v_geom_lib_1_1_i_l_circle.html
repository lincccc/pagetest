Determines whether this circle intersects with the given line. Return the intersection points if yes.

Parameters
pLineThe input line
Return values
ppIntersectPointsOutput the intersection points, it should be an empty set for not-intersected case, or 1 point for tangent case, or 2 points for intersected case.
pTypeReturn the intersection type depends on the count of intersection points which is on the line segment, Bounded (2), Partial (1) or Unbounded (0).

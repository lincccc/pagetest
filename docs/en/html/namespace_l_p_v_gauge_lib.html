enum LPVGaugeDrawFlags

This enumeration represents the flags to control the drawing behavior of the gauging result.

Enumerator
LPVGaugeDrawResult 

Draw the result shape, such as the line or circle.

LPVGaugeDrawSampleRegion 

Draw the sample regions

LPVGaugeDrawSamplePoints 

Draw the valid sample points

LPVGaugeDrawSampleIndex 

Draw the index of the sample region, near its top-left corner.

LPVGaugeDrawDefault 

Default draw flags, which draws only the result shape

LPVGaugeDrawAll 

Draw a gauging result with all components

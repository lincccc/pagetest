Detailed Description

LPV Locate Library, provides classes for classic object/edge location tasks.

This library provides classes for classic object/edge location tasks. To include the definitions of the library's classes, use the following directive:

C++:

#include "LPVLocateLib.h"

C#:

using LPVLocateLib;

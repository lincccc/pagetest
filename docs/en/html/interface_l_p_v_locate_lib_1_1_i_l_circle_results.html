Detailed Description

This interface holds a collection of circle detection results.

Usually you get the LCircleResults object from the ILCircleDetector::Detect() as detection result. You can then use this interface to access individual result and draw the results on graphics windows.

C++:

CLCircleResults m_circle_results;
LPVErrorCode err = (LPVErrorCode)m_circle_detector.Detect(m_img, m_roi, (LPDISPATCH*)&m_circle_results);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_circle_results->Count();
for (int i = 0; i < c; ++i) {
LCircleResult result = m_circle_results->Item(i);
// handle each circle result
}

Detailed Description

This interface provide basic functionalities of the gauge object.

This is a base interface for all kinds of gauge.

To use this interface, you should create a gauge object, such as:

ILCalibNPoints, ILCalibPinHole, ILCalibFFD, ILCalibCustom...
